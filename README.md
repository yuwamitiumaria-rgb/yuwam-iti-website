## Files Included

- `index.html` - Main HTML file (flag removed)

- `logo.png` - Logo image
- All other assets from the original website

## New Color Scheme

- **Header**: Dark gray gradient (#2d3748 to #1a202c)
- **Hero Section**: Light gray gradient (#f5f5f5 to #e8e8e8)
- **Primary Color**: Dark gray (#1f2937)
- **Secondary Color**: Orange (#f97316)

## How to Use

1. Replace the old `index.html` and CSS files with these new files
2. Keep all image files (logo.png, etc.) in the same directory
3. Upload to your hosting (Netlify, GitHub Pages, etc.)

## Customization

You can further customize the colors by editing `style.css`:

- Change `--primary-color` for main text/heading colors
- Change `--secondary-color` for button colors
- Modify gradient colors in `.navbar`, `.hero`, etc.
